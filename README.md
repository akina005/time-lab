# 🕒 Time Lab — 时间实验室  
由 **akina005** 构建的一个可部署到 GitHub Pages / Cloudflare Pages 的静态网页项目。  
它展示当前时间、世界时区、每日进度条、年度进度条，并支持自定义倒计时。

## ✨ 功能特点

### 🕙 实时时钟
- 显示本地时间  
- 支持北京 / 伦敦 / 纽约 / 东京时区切换  
- 秒级自动更新  

### 🌍 世界时间
一次性查看世界四个城市的当前时间。

### 📅 时间进度条
- 今天已过百分比  
- 今年已过百分比  

### ⏳ 自定义倒计时
- 设置日期与时间  
- 支持未来和过去的时间差显示

## 🚀 部署到 GitHub Pages

1. 上传项目到 GitHub 仓库  
2. 打开仓库 → **Settings**  
3. 左侧选择 **Pages**  
4. Source 选择：
5. 保存后等待几分钟，GitHub 会自动生成访问链接  

---

## 🚀 部署到 Cloudflare Pages（可选）

1. 登录 Cloudflare  
2. 创建 Pages 项目  
3. 选择 **Direct Upload**  
4. 上传项目文件夹（包含 index.html）  
5. 发布后即可获得在线访问链接
