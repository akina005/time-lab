# 🕒 **Time Lab — 时间实验室**  
由 **[akina005](https://github.com/akina005)** 构建的一个可部署到 **GitHub Pages** / **Cloudflare Pages** 的静态网页项目。  
它展示了当前时间、世界时区、每日进度条、年度进度条，并支持自定义倒计时。

## ✨ 功能特点

### 🕙 实时时钟
- 显示 **本地时间**  
- 支持 **北京** / **伦敦** / **纽约** / **东京** 时区切换  
- 秒级自动更新  

### 🌍 世界时间
- 一次性查看 **四个世界主要城市** 的当前时间。

### 📅 时间进度条
- 显示 **“今天已过去百分比”**  
- 显示 **“今年已过去百分比”**  

### ⏳ 自定义倒计时
- 支持 **选择任意日期与时间**  
- 能显示未来的 **倒计时**  
- 也能显示 **已过去的时间**

## 🚀 部署到 GitHub Pages

1. 将项目上传到 GitHub 仓库（当前仓库名：`time-lab`）  
2. 打开仓库 → **Settings**  
3. 左侧选择 **Pages**  
4. **Source** 选择：


5. 保存后等待约 30 秒，你的在线网站将在以下地址可访问：https://你的用户名.github.io/你的仓库名/


---

## 🚀 部署到 Cloudflare Pages（可选）

1. 登录 **Cloudflare**  
2. 创建 **Pages** 项目  
3. 选择 **Direct Upload**  
4. 上传整个项目文件夹  
5. 发布后即可获得在线访问链接
